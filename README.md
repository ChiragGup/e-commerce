# E-Commerce Web Application (Next.js + Context API)

A frontend e-commerce web application built using **Next.js** that demonstrates real-world cart functionality and global state management using the **React Context API**.  
The project focuses on clean UI, reusable components, and core business logic commonly used in modern e-commerce platforms.

---

## Features
- Product listing with dynamic UI
- Add items to cart
- Remove items from cart
- Update product quantity
- Global cart state management using Context API
- Responsive design using Tailwind CSS

---

## Tech Stack
- Next.js
- React
- Context API
- Tailwind CSS
- JavaScript
